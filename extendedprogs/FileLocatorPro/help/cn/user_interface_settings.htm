用户界面设置

 Top  Previous  Next

 

用户界面设置控制了FileLocator Pro的“外观和感觉”。

 

 

 

默认

 

将FileLocator Pro配置为采用“标准主题”及选项卡式用户界面。

 

 

经典

 

将FileLocator Pro配置为在外观和行为上与FileLocator Pro version 5.x相似。 Pro version 5.x采用了“经典主题”及MDI用户界面。

 

 

自定义

 

用户所指定的主题，可选择是否采用选项卡式搜索。

 

要更改文件打开的默认设置，请使用 编辑器设置 。
