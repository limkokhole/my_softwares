Extensions

 Top  Previous  Next

 

By default FileLocator Pro opens files in their raw binary format. While this works well for text based formats other formats, such as ZIP files or MS Office 2010 files, do not work as well. Extensions help FileLocator Pro in processing non-text based formats so that their underlying data is correctly retrieved and searched.

 

 

Extensions are organized by:

•IFilters - filters based around Microsoft's indexing service filter specification.
•UTF-8 Default Format - file types that should, by default, be treated as UTF-8.
•Compressed Document Formats - file types that are stored as ZIP files.
•FileLocator Pro Extensions - filters based around Mythicsoft's in house filter specification.
