Search Wizard

 Top  Previous  Next

 

Search Wizard walks the user through entering the basic search criteria with examples and performance tips.

 

The user is presented with three pages asking for the following pieces of information:

•        Name of the file.
•        Contents of the file.
•        Location of the file.

 

On the final page if the 'Start the search automatically' check box is checked the search will begin when the user clicks Finish.
