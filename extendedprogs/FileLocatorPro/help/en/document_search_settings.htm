Document Search Settings

 Top  Previous  Next

 

The Document Search Settings is opened from the Options tab.

 

 

Note: The actual extensions listed will depend on each individual computer setup.

 

File formats with registered processing extensions (extensions specifically designed to help search programs process the format) are shown in the list. If either Deep Search or specific extensions are activated then FileLocator Pro will use the registered extension to process the file format prior to searching. If the search fails or the extension fails to provide results then FileLocator Pro will also search the raw file data if Deep Search or 'Also search raw document data of processed files' are specified.
