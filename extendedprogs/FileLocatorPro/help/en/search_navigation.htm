Search Navigation

 Top  Previous  Next

 

Using search navigation it is possible to browse and/or reuse previous searches. Much like the forward and back buttons on a web browser the search navigation buttons navigate backwards or forwards through searches.

 

 

 

The number of searches available for browsing is determined by the Search Navigation settings in the History Settings.
