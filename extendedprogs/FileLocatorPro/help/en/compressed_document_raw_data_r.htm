Compressed Document Raw Data Reader

 Top  Previous  Next

 

When a document extension, such as DOC, is not active FileLocator Pro still tries to read the raw data of the file. Some file types, however, such as Microsoft Office 2007 and Open Office use ZIP to compress the document's data prior to storage, which requires FileLocator Pro to first unzip the document file before searching the raw data.
